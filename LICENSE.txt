This repository is a modification of the original Livox ROS Driver 2 repository. The modifications made in this repository primarily focus on dockerization of the existing code and the addition of a logger to redirect logs to a specific ROS topic. 

Changes made:
1. Dockerization: Improvements to facilitate the usage of the Livox ROS Driver 2 in Docker environments.
2. Logger: Implementation of a logger that directs logs to a ROS topic, providing better integration with ROS-based applications.
